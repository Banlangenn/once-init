<strong style="text-align: center;">🗼 Make Promise Function Executed Controllable.</strong>

封装可控的 `Promise` 函数。

> 让同一个 `Promise` 不会在同一时间内被执行两次，防止发出重复的请求。

> 或者，同时进行多个相同的 `Promise` ，让所有相同的 `Promise` 都只返回最后执行的那个返回值。
